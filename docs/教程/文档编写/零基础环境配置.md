# 第一步

下载 VS Code (如果已经安装，请忽略这一步)

以下是 VS Code 的官网

https://code.visualstudio.com/

如果对安装过程有疑惑，可以搜索相关教程。例如“VS Code 安装教程”或者求助AI。

# 第二步

打开 VS Code，你会发现 VS Code 的界面都是英文，因此我们要对它进行汉化。(如果已经汉化，请忽略这一步)

在 VS Code 的界面中，我们看左侧，左侧最上方是 VS Code 的图标。
然后往下第一个图标是资源管理器，我们需要找到一个“四个方块”的图标，点击它，然后在往右一点位置的EXTENSIONS下面的搜索框输入"Chinese"，直到搜索框下方出现"Chinese (Simplified) (简体中文) ..."这个插件，点击Install进行安装。安装完成后，右下角会出现一个弹窗，其中有一个按钮"Change Language and Restart"，点击它重启 VS Code，重启完成后 VS Code 的界面就变成中文界面了。

# 第三步

下载 LaTex 工具链 (如果已经安装，请忽略这一步)

以下是 LaTex 的官网

https://www.latex-project.org/

进入官网后，找到"LaTeX the product"下的"Getting LaTex"，点击。
找到此界面的与你的计算机对应的系统版本，优先选择"Tex Live"，接着找到安装包，进行下载。

随后运行安装包，自行安装。

# 第四步

打开 VS Code，进入插件界面 (上文提到的“四个方块”的图标)，搜索并安装 LaTex Workshop 插件，用于进行文档编写。

# 第五步

下载 Git (如果已经安装，请忽略这一步)

以下是 Git 的官网

https://git-scm.com/

进入官网后找到含有"Install"关键字的按钮，点击。如果你的计算机系统是Windows，可以直接点击例如"Git for Windows/x64 Setup"的一行字下载安装包，如果不是Windows，需要使用命令安装，不会的话可以搜索教程或求助AI。

随后运行安装包，此时一定要搜索安装教程或求助AI (因为 Git 的安装过程较为复杂！)，按照合适的需求进行安装！(建议在选择默认编辑器时选择"Use Visual Studio Code as Git's default editor")

# dev-sidecar 的安装与使用

dev-sidecar是用于加速Github的访问的工具。

进入 dev-sidecar 的Github界面 (网址: https://github.com/docmirror/dev-sidecar)

找到右侧的"Releases"，点击下方的"Latest"最新版本，根据你的计算机系统选择安装包 (例如是64位Windows计算机，就选择"DevSideCar-x.x.x-windows-x64"安装包)

运行安装包，自行安装。安装成功后就可以打开 dev-sidecar，进行加速了。
